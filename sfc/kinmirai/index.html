こちらはスーパーファミコン版「ライブ・ア・ライブ」攻略です。リメイク版とは若干攻略が異なります。
「ライブアライブ（HD-2Dリメイク・Nintendo Switch / PlayStation4 / PlayStation5 / Steam）の攻略は以下から。
管理人は同一人物です。

基礎知識

近未来編は比較的オーソドックスなRPGスタイル。
情報収集は主人公アキラの特殊能力、他人の心を読むテレパシー（Yボタン）で行うことが多いのが特徴。
アキラの技は物理攻撃と超能力系に分かれ、超能力技の「●●イメージ」は範囲が広いが待ち時間が長く、やや使い方が難しい。
一方で戦闘はザコ戦の場合BREAK DOWNで倒せることが多く、リーダーも弱いことがほとんどなので、BREAK DOWNを狙うだけならかなり楽な部類（→ザコ敵戦闘）。また、強制戦闘の難易度もさほど高くはない。
BREAK DOWNについては、▶戦闘関係の知識の「BREAK DOWN」を参照。
全体マップでのザコ敵相手の戦闘は、仕様上、経験値を稼ぎやすく、レベル上げがしやすい。装備品・アイテムは元となる素材さえあれば▶アイテム改造で確実に入手でき、回復アイテムも時期限定だがイベントで無限に入手できてしまうなど、本作初心者にも比較的易しいシナリオになる。

以上の理由より、初見プレイの際に最初にプレイするシナリオとしておすすめであるが、近未来編からプレイを始めた場合、慣れてきたあたりで、▶当攻略のトップページから、戦闘関係の知識をひととおり読んでいただければ幸いである。

以下、戦闘＆操作についてのTIPS。

心を読む

アキラは特殊コマンドとして、人（など）の方向を向いてYボタンを押すと人の心が読める。
Aボタンでの「話す」とは違う内容が聞ける他、攻略上必要となる場面も多いので、とりあえず誰かと出会ったらYボタンで心を読んで情報収集すること。
ただし、終盤の筑波研究所でのみ、Yボタンで敵を呼ばれる罠が一ヶ所だけある。
なお近未来編では、Yボタンでも文字送りが可能。

全体マップでのザコ敵戦闘

詳しくは▶ザコ敵戦闘に記してあるが、近未来編では全体マップをうろつくクルセイダー（初期状態で6体）と接触すると戦闘になる。つまりシンボルエンカウントである。
この戦闘は、アキラのレベルによって敵パーティが固定になっている。
どの戦闘もリーダーと手下が設定されていて、リーダーを倒すとBREAK DOWNが発生し手下がすべて倒れる。
BREAK DOWNさせるかどうかでもらえる経験値は変化しないが、BREAK DOWNで倒すと、敵はアイテムをドロップしなくなる。
アイテムを入手して▶アイテム改造をしていくと強い装備品や消費アイテムが手に入る。最終編も見越して、できるかぎりBREAK DOWNさせずに倒していきたい。
全体マップのザコ敵はアキラのレベルが7になるあたりからかなり強くなるため、それまでにタロイモをパーティに加えて、ある程度装備を入手しておくと良い。

アキラの技について

超能力の技は攻撃範囲こそ広いもののタメ時間があるので、タメ時間なしの物理攻撃「ローキック」「エルボー」と使い分けが必要。タロイモ加入後は敵パーティによっては超能力で数を減らしていくのを担当していくと良い。
「ローキック」はヒット数2なので「エルボー」よりも威力が高くなるのだが、レベルが低いうちは2ヒットしにくいこと、技レベルの関係などで、近未来編では「エルボー」の方が頼りになる場面も多々ある。
技のタメ時間の仕様・詳細は、▶戦闘関係の知識 > ターン > 技のタメ時間とターン経過を参照。

超能力の中には追加効果（内部的には行動異常という、状態異常と似て非なるもの）がついているものがある。
行動異常は必ず発動するが、技が効いたターンも含めて、敵の行動ターンで（敵レベル/128）の確率で行動異常が解除される。
ただしBREAK DOWNする敵に対しては、発動後に他の誰かにターンが回ってきた時に行動異常が解除されるため、近未来編ではあまり使い勝手が良くはない。
あまり難しく考えず、広範囲にダメージを与えられる技と割り切って使って構わない。
詳細は戦闘関係の知識の▶行動異常及び、▶BREAK DOWNに記してある。

また、▶アイテム改造を利用し、回数無制限で使える攻撃用アイテムを手に入れると戦闘がかなり楽になる。

ブリキ大王について

近未来編は最終盤にて、操作キャラがロボットのブリキ大王に代わる（シナリオ上、アキラが搭乗する）のだが、ブリキ大王にはアキラのステータスや装備などは一切反映されず、技も別物になるので、アキラとは別の独立した味方キャラの扱いになる。
ブリキ大王での戦闘では経験値やアイテムが得られず、ブリキ大王の装備変更も不可能だが、元からステータスが高く、操作する期間も短い。よほどのことがない限りはラストまで問題なく戦って勝っていける。
（一部ウェブサイトで、「ブリキ大王はアキラの強さに比例する」「アキラの装備品の状態異常耐性を引き継ぐ」というような説明を見かけたことがあるが、おそらくこれは間違いである。ブリキ大王のステータスはデータ集のデータでどの場合でも変化せず、最終編でアキラが「ド根性グラブ」を装備していても、ブリキ大王が腕かため状態になることをプレイして確認済みである）

タロイモの成長と技について

序盤でパーティに加わるタロイモは、戦闘で経験値を得ることができず、レベルアップしない。
その代わり、メカ系の敵を倒した時に一定確率で落とすアイテム「強化パーツ」を持ち物欄で使うことにより、タロイモの最大HPが＋10される。
「強化パーツ」は、ほとんどのメカ系の敵が落とすが、BREAK DOWNで倒すと落とさないため注意が必要。
また、レベルアップしないことから、強化できるのは最大HPのみ、他は装備品で補強するしかないので、近未来編終盤では戦力としてやや使いにくくなってくる。そのためにも下に記すようにアイテム攻撃可能なアイテムを揃えていきたい。

また、一部の装備品をアクセサリー枠に装備すると、技が追加される。
左装備も可能だが、左に装備した場合は技が追加にならないので注意。つまり最大で5種類の技を追加できる。
対応アイテムは、中盤以降行えるようになる▶アイテム改造で作ることが可能（一部は拾える）。
どの順序で装備しても、技の並び順は以下のようになる。

テレポートについて

近未来編の戦闘では、「逃げる」コマンドが「テレポート」になっている。
「逃げる」と同じ効果で、戦闘から逃げ出せるのだが、一定確率でテレポートが失敗してチビッコハウスの中、台所やトイレといった場所に飛び、ごくまれに妙子が入浴している最中の風呂場にテレポートすることがある。
また、アキラが戦闘不能になり、タロイモや無法松しか残っていない時には通常通りの「逃げる」コマンドになる。
全体マップで敵に挟まれたが、どうしても逃げたいという場合、チビッコハウスへ飛んでくれれば脱出できるので、何度も「テレポート」してみると良い。
ただし、シナリオ進行の都合上、チビッコハウスに入れない時期（カズ誘拐中及び、筑波研究所内を探索可能な間）は「テレポート」が失敗しない仕様になっているようである。

余談になるが、世界の合言葉は森部 様の解析によると、テレポートの飛び先と確率は以下のようになっている。

  • 通常の「逃げる」：1/2（50％）
  • チビッコハウスの台所に飛ぶ：1/6（約16.6％）
  • チビッコハウスのトイレに飛ぶ：1/6（約16.6％）
  • チビッコハウスの風呂に飛ぶ：8/54（約14.8％）
  • チビッコハウスの風呂（妙子入浴中）に飛ぶ：1/54（約1.9％）

ちなみに妙子入浴中に風呂にテレポートすると妙子は風呂から飛び出すが、序盤、妙子が洗濯機の前に居る時にこのイベントが発生した時、風呂から出た時に妙子は平然と洗濯を続けていたりもする。

チャート

重要なフラグ（攻略に必要な行動）を記してある。

  • ▶1. 公園

    1. 名前を決める。公園の右側から出ていこうとするとイベント。
    2. ヘルメットの男たちに囲まれる。左の男に向かってYボタンで心を読む。
    3. アキラ＆無法松がクルセイダー3人＆クルセイリーダーと戦闘になる。
    4. 戦闘後、会話の後にYボタンで無法松の心を読むと、自動的にチビッコハウスに移動。

  • ▶2. チビッコハウス

    1. ハウス内の右、テレビ左下に居るワタナベに話しかけ、薬を取ってくるよう頼む。
    2. 玄関左の部屋にいるワタナベに話しかける。
    3. 戻ってテレビを調べるとイベント。ワタナベから「ファーストエイド」をもらう。
    4. ワタナベに話しかけるとカオリについての話が聞ける。
    5. 建物一番左の部屋にいるカオリにYボタン、心を読む。

  • ▶3. 壽商会～タロイモ加入

    1. 壽商会に行き、壽商会の入り口のほぼ真上にある扉の前でYボタンを押す。
    2. 地下に行く藤兵衛を追いかけ、地下の転送装置の上に乗るとイベント。藤兵衛に話しかける。
    3. チビッコハウスに戻りカオリに話しかけ、部屋左下を調べてコンセントを刺す。
    4. 藤兵衛に話しかけてから、心を読む。
    5. タロイモの心を読んで、パーティに加える。
    6. 子供部屋のワタナベに話しかける（アイテムを渡すかどうかは不問）。
    7. 風呂場前にいる妙子に話しかけると洗濯物を干しに行く。
    8. トイレに行き便座に座るとワタナベが来てアイテムをくれる。「妙子のパンチ」がもらえるまで、便座に座ることを4回繰り返す。

  • ▶4. ブリキ大王起動～港

    1. 壽商会の地下11階の藤兵衛に話しかけ、選択肢で「ああ」を選ぶ。
    2. 1階に戻って藤兵衛に話しかける。※攻略上、ブリキ大王を起動イベントを起こす必要はない（やらないとアイテム改造ができなくなるデメリットがある）。
    3. 以上を済ませてチビッコハウスに戻り、下駄箱の横を通りかかると、悲鳴が上がる。
    4. 外に出て妙子の心を読むと、クルセイダー1体と戦闘。自動的に港へ移動。
    5. カズを追いかけ、最後に埠頭に行き、無法松に近づくとイベント、クルセイリーダー＆クルセイダー3体と戦闘。
    6. 戦闘後、クルセイダーズの心を読んでから無法松に話しかけ、更に無法松の心を読む。
    7. チビッコハウスに戻り、左の部屋で、カオリの心を読む。

  • ▶5. 筑波研究所

    途中、無法松と合流しつつ進む（合流箇所は2ヶ所あるが、いずれで合流しても構わない）。
    最奥の部屋で研究員の心を読み、ID番号を入手したら、ID番号で開く部屋に入って、ボスの液体人間Ｗ１号と戦う。

  • ▶6. チビッコハウス～ラストバトル

    1. 壽商会で藤兵衛の心を読む。
    2. 酒場に行き、無法松の心を読む。酒場から出るとチビッコハウスが火事になる。
    3. チビッコハウスに入り、園長に話しかけ建物の中に入る。
    4. 妙子に話しかける、全ての部屋に出入りする（風呂やトイレも含む）、を行うと、ひとつだけ火が邪魔して入れなかった部屋（子供たちが昼寝していた部屋）に入れるようになるので、中に入りカオリの心を読む。イベント後に、ブリキ大王のコクピット内に移動する。
    5. 会話後、行動できるようになったら、無法松の心を読む。
    6. イベント後に操作キャラがブリキ大王になる。強制戦闘を行いつつ北の御出居寺へ向かう。
    7. 御出居寺でラストボスの隠呼大仏戦。勝つと近未来編エンディング。
