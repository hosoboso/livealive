近未来編 ザコ敵戦闘

近未来編では、全体マップをうろつくクルセイダー（初期状態で6体）と接触すると戦闘になる。つまりシンボルエンカウント。
6体はそれぞれ動きが異なり、3体はアキラを追いかける動き（一歩ずつ移動/普通/素早い）、残り3体はランダム移動（一歩ずつ速い移動/一歩ずつ移動/普通）。
クルセイダーの数は、任意のマップに切り替えると元の6体に戻る。

アキラのレベルにより出てくる敵パーティは固定。
クルセイダーまたはクルセイダーＲＳがLEADER、他のメカ敵はBREAK DOWN対象で、LEADERさえ倒せばメカ敵も倒せる。
だがBREAK DOWNで倒すと、敵はアイテムをドロップしない。改造で武器になる「ナパーム弾」やタロイモのHP強化に使える「強化パーツ」は便利なので、できるかぎりBREAK DOWNさせずにすべての敵を倒していくのをおすすめする。
以下ではBREAK DOWNを避けて敵パーティを倒す方法を紹介する。

危険だと思ったら「テレポート」で逃げれば確実に逃げられるが、マップは狭いので、シンボルエンカウントである以上、すぐに再び敵と接触し戦闘になってしまう。ただし「テレポート」は1/2の確率でチビッコハウスに飛ばされることがあるため、しつこく「テレポート」し続けることにより、マップでハマりになることはない。

なお、タロイモは無属性以外の15属性に対する遠距離への反撃技「のうてんショート」（縦横斜3マス、側面へも反撃可能）があるため、リーダーからの攻撃に反撃し、意図せずBREAK DOWNが起きてしまうことがある。タロイモ加入後は、配置に注意すること。

✔ ドロップアイテム一覧
  • クルセイダー：マタンゴライト / ナパーム弾
  • クルセイダーＲＳ：ナパーム弾 / マタンゴライト / へんたいパール
  • BREAK DOWNの敵：強化パーツ

アキラのレベル7あたりから、目に見えてメカ敵が強くなる。
レベル6までのザコ戦でアイテムを集めておき（改造で武器になる「ナパーム弾」は入手できたら使わず取っておく）、アイテム改造が可能になったら装備品を揃え、「ナパーム弾」から強力な攻撃武器「昭和ヒヨコッコ砲」を改造で作っておけば、アイテム使用で出る技「ヒヨコ弾」により戦闘が格段に楽になる。
「昭和ヒヨコッコ砲」は、誰かが使っている待ち時間の間は使えなくなるため、2つ用意すればアキラとタロイモ両方で同時使用ができる。
筑波研究所で無法松が加わっている間を見越して3つ作っておいても構わないが、無法松がパーティに加わっている期間は短いので、面倒なら必要なし。
以降にはBREAK DOWNせずに倒す方法を記しているが、レベル7以降は「昭和ヒヨコッコ砲」を所持している前提で倒し方を書いている。身も蓋もないが、ほぼどの敵も「昭和ヒヨコッコ砲」で倒せてしまうバランスとなっている。

なお、マップ上の戦闘では、レベル13以上だと経験値がもらえなくなる。近未来編で更にレベルを上げたいのなら、筑波研究所のトイレなどで無限湧きのザコを倒すしかない。
ただし、近未来編でそこまでレベル上げをする必要はない。個人的な趣味や、最終編での敵のレベル調整目的で上げるのなら別。
最終編でアキラをパーティに加える時も、戦闘を避ける選択肢で仲間にできるので、レベルを上げすぎてパーティに加えるのが難しくなるような事態も起こらない。

レベル4

  • LEADER：クルセイダー（HP: 10）
  • BREAK DOWN：ラジコン１号（HP: 32）×３

獲得経験値：行動有40、行動無7

まず、「マザーイメージ」でラジコン１号×３だけを攻撃。次に「マザーイメージ」で全体を巻き込み攻撃すれば、一気に倒せる。
基本的に、「マザーイメージ」を初期位置で2回使えばOK。2回目の「マザーイメージ」のタメ時間にクルセイダーが近寄ってきて、勝手に「マザーイメージ」に巻き込まれてくれる。

レベル5

  • LEADER：クルセイダー（HP: 10）
  • BREAK DOWN：ラリアット１号（HP: 72）×２

獲得経験値：行動有40、行動無6

敵の配置上、「マザーイメージ」「スリートイメージ」などではクルセイダーを巻き込み一発でBREAK DOWNとなってしまうため、「エルボー」でラリアット１号を叩く。初期装備でも、正面からの「エルボー」3回、または背面から「エルボー」2回で倒せる。
ラリアット１号の攻撃は、大したダメージにはならない。
2体のラリアット１号を撃破したら、「エルボー」など適当な技でクルセイダーを撃破。

レベル6

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：ラジコン２号（HP: 26）×１４

獲得経験値：行動有40、行動無6

「マザーイメージ」の範囲にラジコン２号を入れて連発していく。クルセイダーＲＳには「マザーイメージ」が当たってもダメージ0なので、「マザーイメージ」に巻き込んでも倒してしまう心配がない。
ダメージにブレがあるが、最低2回「マザーイメージ」を当てればラジコン２号を倒せる。
残り少なくなったら「エルボー」「ローキック」で倒していっても良い。
最後にクルセイダーＲＳを倒すが、残り一体になると近寄ってきて攻撃してくる（斜め方向は3マスの遠距離攻撃があるので注意）。
アキラが耐えられる程度のダメージしか与えてこないため、気にせず「エルボー」「ローキック」で撃破。
なお、ラジコン２号は隣接すると「爆発」で火炎フィールドを作った後に自滅する性質があるので気をつけること。ラジコン２号はこの「爆発」以外の技を持っていない。

レベル7

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：キョクシン１号（HP: 120）×３

獲得経験値：行動有40、行動無6

キョクシン１号が強いため、倒すのならばタロイモ加入後、アイテム改造で装備品を整えてからが良い。
倒し方としては、

  • アキラは「スリートイメージ」でキョクシン１号たちの向きを変える
  • タロイモは自分の方向を向いていないキョクシン１号を叩く

を繰り返す。
アイテム改造で「昭和ヒヨコッコ砲」を入手していれば、キョクシン１号はこれ一発で倒せる。「昭和ヒヨコッコ砲」は、アイテム欄においておけばアキラでも使える。

レベル8

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：ラリアット１号（HP: 72）×２　ラリアット２号（HP: 104）×２

獲得経験値：行動有40、行動無7

「昭和ヒヨコッコ砲」さえあれば苦労しない。どの敵も一撃で倒せる。
「昭和ヒヨコッコ砲」がない場合は「ヘルイメージ」などで地道に削るが、装備が整っていればアキラの「エルボー」「ローキック」の威力も上がっているので、アキラとタロイモで殴っていくのも手。HPの低いラリアット１号（オレンジ色の敵）から倒していけば楽。

レベル9

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：ブル・コマツ１号（HP: 288）　ラジコン２号（HP: 26）×４

獲得経験値：行動有40、行動無6

ブル・コマツ１号の「セメント攻撃」（横斜1～2マス）は石化攻撃。メカのタロイモも石化無効効果は持っていないし、近未来編ではアイテムの「ポーションＮＯ９」でしか石化を治せない。誰かが石化したら逃げることも考えた方が良い。
アキラは「マザーイメージ」の攻撃範囲にラジコン２号全てが収まるように移動してから、「マザーイメージ」でラジコン２号を倒す。
「マザーイメージ」の間に、タロイモが「昭和ヒヨコッコ砲」でブル・コマツ１号を攻撃。
ブル・コマツ１号はHPが高く、「昭和ヒヨコッコ砲」と「マザーイメージ」で倒しきれない場合は、更に適当な攻撃を加えて撃破。
（「昭和ヒヨコッコ砲」の追加効果で敵の体の値が上がってしまうと、倒しにくくなる可能性あり）
後はクルセイダーＲＳを倒して終了。

レベル10

  • LEADER：クルセイダー（HP: 10）
  • BREAK DOWN：ラリアット１号（HP: 72）×3　ラリアット２号（HP: 104）×２　ラリアット３号（HP: 128）×２

獲得経験値：行動有40、行動無7

ラリアットシリーズの反撃が痛いため、できるだけ一撃で倒していきたい。
アキラは「フレームイメージ」が入れば１号と２号を一撃で倒せる（装備次第ではローキックでも一撃で倒せるようになっているはず）。タロイモは「昭和ヒヨコッコ砲」で３号を攻撃。
クルセイダーの移動の邪魔になっている画面左側の敵3体を倒すとクルセイダーが接近してくるのだが、タロイモが攻撃されると反撃でクルセイダーを倒してしまうので、BREAK DOWNを避けるためにも配置に注意。
また、ラリアット１号の「大旋風」で向き変えされると、技や「昭和ヒヨコッコ砲」をタメている場合は解除されて面倒なので、ある程度倒したところでラリアット１号に隣接した方が良い。「大旋風」はラリアット１号から離れていると出してくる。

レベル11

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：ラリアット３号（HP: 128）　へーベル１号（HP: 144）×７

獲得経験値：行動有40、行動無7

横に並んでいるへーベル１号7体は、「ヘルイメージ」で吹き飛ばしつつ攻撃が一番効率が良い。「ヘルイメージ」2発で片付く。
タロイモはラリアット３号を「昭和ヒヨコッコ砲」で倒すが、最初の「ヘルイメージ」で吹き飛ばされる可能性があるので、ラリアット３号の左側を狙うこと。左側を狙えば吹き飛ばしで移動後でもラリアット３号に命中する。
「ヘルイメージ」待ち時間でへーベル１号が範囲外に移動してしまった場合は、「昭和ヒヨコッコ砲」で始末する。
後は残ったクルセイダーＲＳを撃破。

レベル12

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：サイコ１号（HP: 176）×３

獲得経験値：行動有40、行動無8

サイコ１号を「昭和ヒヨコッコ砲」で近い順に倒していけばOK。「昭和ヒヨコッコ砲」なら一撃。
最後に残ったクルセイダーＲＳを撃破。

レベル13以上

  • LEADER：クルセイダーＲＳ（HP: 24）
  • BREAK DOWN：ラジコン２号（HP: 26）×７　へーベル１号（HP: 144）×６　キョクシン２号（HP: 128）

まずクルセイダーＲＳ以外を「シャドウイメージ」の範囲に入れるようアキラが移動、「シャドウイメージ」でラジコン２号、二度目の「シャドウイメージ」でへーベル１号を全て倒す。
キョクシン２号はやたら硬く、「シャドウイメージ」のダメージも含めて、「昭和ヒヨコッコ砲」を2～3発当てないと倒せない。タロイモはひたすら「昭和ヒヨコッコ砲」をキョクシン２号に当て続ける。
最後に残ったクルセイダーＲＳを撃破。
ちなみにキョクシン２号のステータスは、近未来編で最強といってもいい数値（力230/速200/体180/知200）

※これ以降は経験値をもらえない。アイテムは落とす。
