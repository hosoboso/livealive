11. リフレッシュ・ルーム / 2階（レベル2）

部屋左上のコーヒーメーカーを調べて「コーヒー」を作り、ヒューイに渡す。
すでに「コーヒー」を所持しているのならそれを渡せば良い。
イベント後、全員出ていく。

12. カークの部屋 / 2階（レベル2）

カトゥーの部屋の斜め左上にある、カークの部屋に行く。
扉の左のコンピュータを調べ、名前を登録し中へ。
部屋の中のカトゥーに話しかけ、部屋を出る（攻略上、伍長に話しかける必要はない）。

メモリーカード

この時点からカークの部屋に出入りできるようになるが、ベッドの横にあるコンピュータを調べると、合言葉を聞かれる。
合言葉のヒントはカトゥーとの会話で聞けるが、正解は「ワープ」である。
「ワープ」と入力すると、あるメッセージが表示され、ベッドの下側を調べる（ベッド左下マスに立ち右を向いてAボタン）と、アイテム「メモリーカード」が手に入る。
「メモリーカード」を所持した状態で、リフレッシュ・ルームのゲーム「キャプテンスクウェア」を調べると、メモリーカードをインサートするかどうか聞かれる。
インサートしてゲームを開始すると、途中でゲームオーバーになっても、再プレイの時にメモリーカードをインサートした時、ゲームオーバーになったステージからプレイ可能になる。
ゲームをクリアしたメモリーカードだと、また最初のステージからのスタートとなる。
なお、「キャプテンスクウェア」の左側のコンピュータを調べると、メモリーカードを初期化するかどうか聞かれる。初期化すると最初のステージからプレイ開始になる。
キャプテンスクウェア攻略

13. エアロック / 1階（レベル1）

ヒューイと話す。

14. 医務室 / 2階（レベル2）

順不同で12. と13. を行ってから、医務室へ行くと、カークの遺体がなくなっている。
医務室の奥まで行く必要はない。医務室に入って音楽が変わった時点でフラグが立つ。
カークの部屋のカトゥーか、エアロックのヒューイに話しかけ、カークの遺体のことを伝える。医務室からなら同じフロア内のカークの部屋の方が近い。

なお、医務室でカークの遺体がなくなっていることを確認してから、次の15. を行うまでは、それまでにレイチェルの部屋に入ったことがあったとしても、一時的にレイチェルの部屋に入れなくなる。

15. 医務室 / 2階（レベル2）

医務室に戻るとイベント。
自動的にレイチェルの部屋へ移動する。

16. レイチェルの部屋 / 2階（レベル2）

イベント、レイチェルが飛び出し、皆追いかける。

17. エアロック / 1階（レベル1）

エレベータで1階に着いた時点で、念の為セーブをしておくこと。
エアロックにこの時点では行く必要は攻略上ないが、入るとカトゥー、ヒューイ、レイチェル、伍長の4人がいて言い合いをしている。
部屋の外に出ると物音がする。

18. べヒーモス倉庫 / 1階（レベル1）

17. をしてもしなくても、ベヒーモスがいた倉庫に入ると、ベヒーモスがいなくなっている（音楽が変わる）。
べヒーモスがいた倉庫は、画面上では見えないが上側からも入ることができる。
上側、下側どちらの出入り口から入っても構わない。
倉庫から出るとベヒーモスがうろついており、接触するとゲームオーバーになるので、ダッシュして上方向の扉に入る。
ベヒーモスの移動速度は、キューブの通常の歩行速度と同じなので、ダッシュならば引き離すこともできる。
なお、倉庫の上側から倉庫に入って出た場合、画面上にベヒーモスの姿は見えないが（そのうち画面下から走ってやってくる）、ベヒーモスの姿を見ないまま、上方向の扉に入ってしまっても構わない。

ベヒーモス

べヒーモスが逃亡した後は、船内をベヒーモスがうろつくようになり、隣接しただけでゲームオーバーになる。
以降、安全な場所でこまめにセーブしながら進めることをおすすめする。べヒーモスに追いかけられている最中のセーブは詰み防止のために避けた方が良い。
ベヒーモスが出現するのは1階の倉庫フロアと2階の通路だけで、室内やエレベータ内に出現することはなく、逃げ込めば追いかけてくることはない。
また、ベヒーモスの居場所はストーリー進行により固定で、ランダムで色々な場所に出現するという訳ではない。
この先は、ベヒーモスの居場所もできるかぎり併記してある。

19. エアロック / 1階（レベル1）

レイチェル以外の誰かに話しかけて、ベヒーモスの件を伝える。
エアロックが開くイベント後、自動でリフレッシュ・ルームに移動する。

20. リフレッシュ・ルーム / 2階（レベル2）

レイチェルにコーヒーを渡す。
会話の後に、レイチェル、続いてヒューイが部屋を飛び出す。
そのまま10秒程度待つと、カトゥーと伍長が部屋を出ていってしまう（その前にキューブが部屋を出ようとした場合、カトゥーに止められる）。
カトゥーたちを追いかけて部屋を出て、すぐ右の通路を下に向かうと、ヒューイたちが倒れている。
なお、この時はコールドスリープルーム真横の通路にベヒーモスが居るので近づかないように。
イベント後に、カトゥーに端末室に行くよう頼まれる。
セーブをしてから向かうこと。

21. 端末室 / 2階（レベル2）

端末室前の通路には、ベヒーモスがうろついている。キューブに向かって接近してくるので、ひとつ前のフロアでセーブをし、うまく誘導する。
まず真横（左）に引き付けて、近寄ってきたら、ぐるりと時計回りするように上、右、下と逃げて右に向かい端末室に入ると良い。
うまくできない時はダッシュより歩きで誘導した方が良いかもしれない。その方がベヒーモスの動きがわかりやすい。

端末室に入ったら、中のコンピュータを調べる。パスワード「OAKFDE」が表示される（プレイヤーが覚える必要はないが、メモしておくと最終盤のおまけ要素で役立つ場面がある）。
パスワードを聞いた後は、端末室前の通路からベヒーモスがいなくなるが、コールドスリープルーム南側入り口のフロアに移動している。一度セーブしてから移動すること。
また、この時はリフレッシュ・ルーム北側の通路が閉じてしまっており通れない。2階からでは大回りしてコールドスリープルーム北側入り口に行くこともできなくなっている。
ただし、南エレベータで1階や3階に移動することはできるので、1階または3階に移動し、北エレベータを使って2階に降りて、コールドスリープルーム北側入り口へ向かえば、ベヒーモスに出くわすことなくコールドスリープルームに行くことが可能。

※ヒューイが亡くなってからは、ヒューイの部屋にも名前登録後に出入りできるようになる。
室内のコンピュータを調べることも可能。

※「OAKFDE」を聞いた後、次の22. を行う前に船長室前に行って、扉の横を調べるとパスワード入力画面になるが、「OAKFDE」と入力してもエラーになり入れない（何を入力しても入れない）。
これをおこなってからコールドスリープルームに行くと、カトゥーとの会話内容が少し変化する。その後の展開は同じ。

22. コールドスリープルーム / 2階（レベル2）

コールドスリープルーム南側入り口のフロアにはベヒーモスがいる上、上側へ行く通路を遮断されてしまう。ここも21. と同じように誘導してから扉に入る。
また、21. で述べた通り、大回りしてコールドスリープルーム北側入り口からコールドスリープルームへ入っても良い。
部屋に入ると会話、自動的に船長室の前まで移動する。
船長室の前で「パワージャッキ」を渡され、持ち物に加わる。
船長室に入る。

「パワージャッキ」は使用回数が10回に限られているので注意。使いすぎると、場合によっては詰む。
セーブデータのスロット数に余裕があるなら、この時点でのセーブデータを別に保存してから進めるのも良い。
「パワージャッキ」を使うと新たに、非常用通路に入れるようになるのだが、「パワージャッキ」の使用回数制限を考えた場合、無闇に入るべきではない。このページ下の非常用通路についても参照。
また、「パワージャッキ」使いすぎで詰んでしまった場合はＳＦ編を最初からやり直すしかない。
ＳＦ編トップページの「パワージャッキ」の説明に従い、「シナリオを選び直す」でやり直すこと。

23. 船長室 / 2階（レベル2）

船長室内で会話のみ。

24. 伍長の部屋 / 2階（レベル2）

船長室の会話後、船長室を出て下へ向かい、カトゥーたちの部屋入り口のあるフロアに行くと、警報音が鳴っている。
伍長の部屋（カトゥーの部屋の左）の扉の右側のコンピュータを調べて、名前入力して中に入り、部屋内の左上のコンピュータ（出入り口のすぐ左）を調べる。
機密文書が表示される。

なお、「パワージャッキ」入手後から、伍長の部屋の探索の最中は、コールドスリープルーム北側入り口フロアにベヒーモスがいるので注意。

25. 船長室 / 2階（レベル2）

船長室に入る前にセーブしておくこと。
カトゥーに話しかける。カトゥーとともに部屋を脱出。
すぐにべヒーモスに体当たりされ、カトゥーとはぐれてしまう。
ここからカトゥーと再会するまで、キューブ一人でベヒーモスに追われることになる。
ベヒーモスと接触したり、通路の水が吹き出す罠に一定時間かかると、即ゲームオーバーとなる。
初見では事故ってゲームオーバーになっても仕方がないので、船長室に入る前にセーブすること。

やらねばならないことは以下の通り。

  1. カトゥーとはぐれた直後、下からベヒーモスが追いかけてくる。通路を上に進むと扉が閉じていて閉じ込められるので、扉をAボタンで調べ「パワージャッキ」を使い（ここは攻略上、「パワージャッキ」を使うしかない）、扉を開いて逃げる。
  2. 1階倉庫フロアに入って、ベヒーモスの姿を確認する。
    1階の北側から入ると、真正面からベヒーモスが突進してくる。南側から入ると、上方向に向かう途中で左側からベヒーモスが飛び出してくる。
  3. 3階の通路の罠を抜ける。罠は以下のa. かb. のいずれかを抜ければ良い。
    1. メインコンピュータルームの下の通路を上から下へ抜けて、水が吹き出す罠を見る
    2. メインコンピュータルームの上の通路を下から上へ抜け、閉じ込められたら「パワージャッキ」で開く
  4. 2. と3. を順不同で済ませると、2階でキューブそっくりのロボットが逃げていくのを目撃する。
    （2階北方面から来た場合はコールドスリープルーム南側入り口、2階南エレベータから来た時は南エレベータを出たフロア）
  5. キューブそっくりのロボットを追いかけて、カトゥーの部屋へ向かう。

ルートは比較的自由で、北エレベータ前フロアの2マス右（▼マークのあるところ）に向かって上ボタンかAボタンで持ち物欄を出し、「パワージャッキ」を選択すると現れる非常口を使っての移動もできるようになる。
非常用通路にはベヒーモスが出現しないし、追いかけてもこない。
以下は解法の一例。ダッシュのみで抜けられ、「パワージャッキ」の使用回数も最小限で済む、比較的楽なルート。

  1. まず追い出された先の通路を真上へ。ベヒーモスが下から追いかけてくるので上へ向かうが、通路出口は閉じて出られなくなっている。扉をAボタンで調べ、持ち物欄を出し「パワージャッキ」を選択し扉を開いて上へ。
  2. リフレッシュ・ルーム北側出口に出る（引き返さないこと。再び出入り口を塞がれる）、そのまま上に抜けて北エレベータで1階に降りる。
  3. ベヒーモス倉庫のフロアに入る。そのまま3秒程度待っていると、下からベヒーモスがやってくるので、すぐに上キーを押して北エレベータ前に戻る。
  4. 北エレベータを使って3階に登る。3階はひたすら下方向へ。
    メイン・コンピュータルーム下の通路では、出入り口を塞がれてから数秒後に通路下付近で水が飛び出る罠が起動する。
    この通路に入ると同時に、下方向へダッシュし続けていれば水の罠を回避して下へ抜けられるのだが、立ち止まったりして水が吹き出す罠に行く手を阻まれた場合、引き返し、「パワージャッキ」で扉を開けて上に戻る。
    再び下の通路に入った時には罠が初期状態となっているので、下方向ダッシュで抜けること。
    南エレベータから2階へ。
  5. 2階南エレベータを出ると、左上にキューブそっくりのロボットが逃げていくので追いかける。カトゥーの部屋へ。

上の方法だと「パワージャッキ」使用回数は1～2回で済むが、初見プレイだと色々な罠に引っかかり、「パワージャッキ」を更に消費することもあるだろう。
この場面では、使っても5回以内でカトゥーの部屋へ行くようにしたい。
この回数を越えてしまうようなら、リセットして船長室に入る前からやり直す方が無難。
終盤にかけて少なくともまだ2～3回は「パワージャッキ」を使う必要がある。

他にも、最初の襲撃後に2階北エレベータ前で非常口を開いて、ハシゴを使って1階や3階に移動する方法もある。
下の「非常用通路について」も参照。
非常口内部は屋根裏にあたる部分（1階と2階の間、2階と3階の間）に、何もない広い通路が広がっているので迷子にならないよう注意。

ちなみに非常口内部の通路では、一か所、通過するとかすかにベヒーモスの鳴き声がするポイントがある。
また、ニセキューブ出現後にコールドスリープルームに行くと、警告音が鳴っており、レイチェルのカプセルに異変が起きているイベントも見られる。
更に、カトゥーとはぐれた直後から、各所でコンピュータの端末を調べた時のメッセージが変わっている。
攻略上やる必要はない。

非常用通路について

「パワージャッキ」入手直後から、非常用通路（リメイク版におけるダクト）を利用できるようになる。
非常用通路は、各階層の間にあり、各階層の北エレベータ横（マップ画像では「非常口」としている）の▼マークに向かってAボタンまたは上ボタンを押すと持ち物欄が開くので、「パワージャッキ」を選択すると非常口が開いて中に入れるようになる。一度開けた非常口は開いたままになる。
またはすぐ近くの床の扉（格子状のマス、マップ画像では「通風口」としている）から入れ、エレベータを使わなくても各階層を行き来できるようになっている。
2階（レベル2）は北エレベータ近くの他にも、通路の各所に通風口が多数設置してあり、ここからでも非常用通路へ移動可能。
ただし、通風口から非常用通路に入る場合、一方通行なので注意。非常用通路から出たい場合は、エレベータ横非常口を利用するしかない。
通風口は非常口とは異なり、開いたままにならない。落とし穴のような罠になる心配はない。
また、中に入ってから他の階層の非常口で外へ出る時には「パワージャッキ」を使う必要はないし、使用回数も減らない。

通風口に入る方法だが、通風口のマスの真上でAボタンを押しても何も起きず入れない。
通風口のマスの隣で、通風口に向かってAボタンを押すと持ち物欄が開き、「パワージャッキ」を使用すると通風口が開く仕組みである。開くと自動的に内部へ移動するため、試しにやってみる程度ならセーブしてから行うこと。

攻略冒頭からしつこく記しているが、「パワージャッキ」は使いすぎると詰むため注意。
ベヒーモスから逃げるため、止むを得ないという場合の移動方法として使用するのが良いだろう。
ＳＦ編を隅々まで堪能するために探索してみたいというのなら、この先の「パワージャッキ」の使用回数を考慮した上で行くこと。事前セーブも忘れずに。

「3階と2階の間の非常用通路」「2階と1階の間の非常用通路」「1階の地下」に当たる部分には、それぞれ下方向に狭い通路が存在している。
特に「2階と1階の間の非常用通路」は広く、同じような構造が続くので迷わないよう注意。抜け出る場合はとにかく上へ向かえばハシゴに出るはず。
また、「2階と1階の間の非常用通路」では、一か所、通過するとかすかにベヒーモスの鳴き声がするポイントがある。
下のマップ画像でも、鳴き声がする位置を記してある。
鳴き声がするだけで襲われることはないが、ホラーが苦手な人は無闇に「2階と1階の間の非常用通路」を探索しない方が良いかもしれない。
