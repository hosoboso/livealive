西部編 攻略

こちらはスーパーファミコン版「ライブ・ア・ライブ」攻略です。リメイク版とは若干攻略が異なります。
「ライブアライブ（HD-2Dリメイク・Nintendo Switch / PlayStation4 / PlayStation5 / Steam）の攻略は以下から。
管理人は同一人物です。

基礎知識

西部編は、無法者集団クレイジー・バンチからサクセズ・タウンを守るため、襲撃前に街に罠をしかけ、クレイジー・バンチと決闘、という流れ。
罠は街中を探索して探し、街の人に仕掛けてもらう。罠探しと仕掛けるのは時間制限がある。
仕掛けた罠の数に応じて、ラストバトルの敵の数が決まる。罠の数が最大であれば決闘で戦うのは西部編ラストボス1体だけになるが、逆に罠が全て失敗（仕掛けられない）と15人を相手にするハメになる。
戦闘回数はごく少なく、主人公のサンダウンは西部編ではレベルアップしない。ラストバトルを楽にしたいのなら罠をしっかりと仕掛けること。あえてラストバトルの難易度を上げるため罠を仕掛けず挑むという、自由なプレイスタイルも楽しめる。
罠を仕掛けず挑むなら▶15人と戦ってみるを参照のこと。

当攻略では「最終編にできるだけ良い装備を持ち込み、かつ、西部編ラストバトルをできるだけ楽に戦う」ルートでクリアすることを目標として攻略を記している。

（大したことではないが）西部編登場キャラのひとり、マッドドッグは、取扱説明書等では「マッド・ドッグ」表記だが、ゲーム内では「マッドドッグ」（または「マッド」）表記で統一されているため、本攻略でも「マッドドッグ」と表記している。
なお、ゲーム内では「サクセズ・タウン」も「サクセズタウン」表記が混ざっていたりするが、当攻略では「サクセズ・タウン」で統一している。

チャート

重要なフラグ（攻略に必要な行動）を記してある。

  • ▶1. 開始～サクセズ・タウン到着

    1. 名前入力、イベント後にマッドドッグと戦う。
    2. サクセズ・タウンに到着、CRISTAL BAR（酒場）に入る。

  • ▶2. 酒場

    1. カウンター越しにマスターに話しかける。
    2. 会話の後、選択肢で「返す」→「消えろ」でパイクと戦闘。
    3. ビリーとの会話の選択肢は「ウソじゃない」を選ぶ。
    4. マッドドッグが決闘に誘うので、酒場の外へ出る。
    5. 酒場に戻ったら、全員と話をする。全員から話を聞いた後、もう一度マッドドッグに話しかけると、罠を仕掛けることを提案される。
    6. ビリーから「保安官バッジ」を預けられ、マッドドッグがパーティに加わり、罠探しと仕掛けのカウントダウンが開始。

  • ▶3. 罠探し＆仕掛け

    ※罠を探したり仕掛けることは必須ではない。
    鐘が8つ鳴り終わるとクレイジー・バンチが攻め込んでくる。

  • ▶4. ラストバトル

    1. クレイジー・バンチが攻め込んできたら選択肢「行くぞマッド」。
    2. その後の選択肢は「もちろんだ」でラストバトル。「‥‥」だとラストバトル前にもう一度選択肢が出るので「‥‥マッド？」を選ぶと進む。
    3. ラストボスのディオ（と、罠の数次第ではクレイジー・バンチ）とラストバトル。
    4. ラストバトル戦の後、「そうだな」→「ああ」を選択するとマッドドッグと最後の戦い。倒すことも逃げることも可能。
    5. スタッフロール後、マッドドッグ戦で逃げた場合のみ、マッドドッグと再戦になり、倒すとエンディング。
